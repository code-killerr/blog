## 如何快速开始
## 创建一个新文章
``` bash
$ hexo new "文章题目"
```

更多信息: Writing

## 运行本地服务器

``` bash
hexo server
```

或者

``` bash
hexo s
```
在运行服务器时所做出的更改会立即被服务器运用，但有时涉及到config的部分设置需要重启服务器
## 生成网页

```
hexo generate
```
或者

``` bash
hexo g
```

## 提交网页到站点
``` bash
hexo deploy
```

或者

``` bash
hexo d
```
## 一些Markdown语法
### 注意:
Markdown中符号后面记得加上空格，否则不会显示相应效果
### 多级标题
Markdown中具有多级标题，一级到六级，文本前加上#号即可，有多少个#号代表第几级标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```
#一级标题
##二级标题
###三级标题
####四级标题
#####五级标题
######六级标题
```
跟代码有自己的风格一样，#号后面通常加空格是标准的Markdown写法

### 代码框
使用三个`可以实现代码框，注意不是引号，是键盘左上角1旁边的那个点，在’’’后面加空格再加上代码框中代码使用的语法即可生成对应的高亮
效果如下

``` c++
print("hello world")
```
`给你写出来的话加上背景`使用`符号即可
